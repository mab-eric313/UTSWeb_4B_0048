# Debian Nova — Distro Linux curated rolling release berbasis Debian Testing

Referensi desain:
- https://www.suse.com/
- Layout dan konten mengikuti desain website ini

- https://startbootstrap.com/template/small-business
- Sebagai template

Debian Nova menawarkan:
1. Menggunakan Instalasi yang mudah digunakan dengan Calamares GUI
2. Menyediakan dan mendeteksi otomatis Non-Free Firmware
3. Curated Rolling, paket Testing yang divalidasi QA sebelum rilis
4. Menyediakan sistem distribusi aplikasi Flatpak secara default
5. Menyediakan fitur Snapshot dan Rollback dengan file system Btrfs
6. Menyediakan Security backport dari Debian Stable
